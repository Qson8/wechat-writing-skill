# 公众号文章评估框架

## 评估流程

拿到文章后，按以下5个维度打分，再给综合评分和修改建议。

---

## 五个评估维度

### 1. 标题（0-20分）

| 分数 | 标准 |
|------|------|
| 16-20 | 有具体数字或场景，说清楚读者收益，不夸张不焦虑 |
| 10-15 | 基本清楚但不够吸引人，或稍微夸张 |
| 0-9 | 空洞、夸张、或完全看不出文章写啥 |

检查项：
- [ ] 有没有具体数字或场景
- [ ] 能不能从标题判断读者能得到什么
- [ ] 有没有用禁用词（颠覆/逆袭/干货/破局等）
- [ ] 字数是否在25字以内

### 2. 开头（0-20分）

| 分数 | 标准 |
|------|------|
| 16-20 | 从真实场景或经历切入，读者有代入感，自然引出主题 |
| 10-15 | 能引入主题但缺乏代入感，或开头稍显套路 |
| 0-9 | 自嗨式开头，或制造焦虑，或直接说"本文介绍..." |

检查项：
- [ ] 有没有从具体场景/经历切入
- [ ] 前3句有没有让读者觉得"这说的是我"
- [ ] 有没有"今天给大家分享"这类自嗨句式

### 3. 正文内容（0-30分）

| 分数 | 标准 |
|------|------|
| 24-30 | 有具体数据/案例/步骤，每个观点都有支撑，读者看完能做点什么 |
| 15-23 | 内容有价值但部分太虚，或步骤不够具体 |
| 0-14 | 大量泛泛而谈，没有实质内容，或全是鸡汤 |

检查项：
- [ ] 每个观点有没有具体数据或案例支撑
- [ ] 有没有读者可以立刻执行的行动
- [ ] 有没有大量废话或鸡汤
- [ ] 段落是否简短（每段≤4行）

### 4. 语言风格（0-20分）

| 分数 | 标准 |
|------|------|
| 16-20 | 大白话，口语化，读着舒服，没有禁用词 |
| 10-15 | 基本通顺但偶有高大上词汇或夸张表达 |
| 0-9 | 大量营销词/焦虑词/高大上词，读着不舒服 |

检查项：
- [ ] 有没有出现禁用词（逐一检查）
- [ ] 句子是否口语化、短句为主
- [ ] emoji数量是否合理（≤3个）
- [ ] 整体读下来是否舒适自然

### 5. 结尾（0-10分）

| 分数 | 标准 |
|------|------|
| 8-10 | 简单总结+一个行动建议，或提问引发互动，自然收尾 |
| 4-7 | 有结尾但稍显生硬，或总结不到位 |
| 0-3 | 没有结尾，或明显的"求关注求转发"索取式结尾 |

---

## 综合评分

| 总分 | 结论 |
|------|------|
| 85-100 | 可以直接发，优质文章 |
| 70-84 | 小改一下再发，主要问题修掉就行 |
| 50-69 | 需要较大修改，建议重点改正文和语言 |
| 0-49 | 建议重写，框架或风格有根本问题 |
